What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface that represents the structure of an HTML or XML document as a tree of objects. It allows developers to access, manipulate, and modify content and structure dynamically using languages like JavaScript, enabling interactive web applications and enhanced user experiences.

How do you select an element from the DOM?

To select an element from the DOM, use methods like getElementById(), getElementsByClassName(), or querySelector(). For example, document.getElementById('myElement') retrieves an element by its ID, while document.querySelector('.myClass') selects the first element matching the specified CSS selector.

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique in the DOM where a single event listener is added to a parent element instead of multiple listeners on child elements. This approach improves performance and simplifies code, as events bubble up to the parent, allowing for easier management of dynamic elements and reduced memory usage.

How do you manipulate an element's attributes and styles using the DOM?

You can manipulate an element's attributes using setAttribute() or by directly accessing properties, like element.id. To change styles, use the style property, e.g., element.style.color = 'red'. This allows for dynamic updates of attributes and CSS properties directly in JavaScript
